color

color: number
  • Defined in node.ts:70

Node color.

Applies only when type is NodeType.COLOR

Common convention follows Puyo Puyo color cycle:

  • Red = 0
  • Green = 1
  • Blue = 2
  • Yellow = 3
  • Purple = 4

More colors may be specified independently from here.
